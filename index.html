Лаба №1 Создание пользователя для себя, любимого

Задание:
1. Создать пользователя с использованием графической оболочки..
2. Создать пользователя в терминале программой adduser (или useradd).
Предварительно прочесть man.

!!!!!!Все действия выполнялись в дистрибутиве KDE neon 5.12!!!!!

1. Заходим в настройки => Учетная запись => Управление пользователями => Добавить пользователя => Вводим все данные нажимаем применить.

2.Открываем терминал вводим: sudo useradd max
Что бы задать пароль вводим: sudo passwd max
